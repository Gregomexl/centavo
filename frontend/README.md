# Centavo Frontend

Personal expense tracker web dashboard built with Next.js 16 and React 19.2.

## Tech Stack

- Next.js 16 (App Router)
- React 19.2
- TypeScript 5.x
- Tailwind CSS 4.0
- shadcn/ui

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

## Environment Variables

Copy `.env.example` to `.env.local`:

```bash
cp .env.example .env.local
```
